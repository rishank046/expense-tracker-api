# Expense Tracker API and Web Interface
### a expense tracker api that focuses on financial management and a modern User Interface that makes using API easy. 
### This project features a custom-built REST API using Vanilla Node.js and mySql for database management.

# Technologies used
* used JS on NODE.JS
* mySql as database
* fronted techs HTML , CSS , JS

# Starting server

### start local mySql server
* on Linux
```
sudo systemctl start mysqld
```
* on windows
### use XAMPP [download XAMPP](https://www.apachefriends.org/)

### start listning on a port
```
  npm run start
```
